# MIDI Drum Remapper

This is a conversion tool used to import drum MIDI files created in a DAW into score writing software like MuseScore **without issues**.

## Installation

Please download `MIDI Drum Remapper.zip` from [Release](https://github.com/nyoro-wrl/MIDI-Drum-Remapper/releases) and unzip it.

## Usage

Please check [here](assets/README.md) or check `README_en.md` in the unzipped files.
